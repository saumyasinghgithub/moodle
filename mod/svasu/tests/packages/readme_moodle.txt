Description of test packages
============================================

Sample Packages downloaded from http://svasu.com/svasu-explained/technical-svasu/golf-examples/

* singlescobasic.zip - Single SCO with basic runtime calls. SVASU 1.2.
* singlesco_svasu12.zip - Single SCO content packaging example. SVASU 1.2.
* RuntimeMinimumCalls_SVASU12.zip - Multi-SCO packaging example. SVASU 1.2.
* RuntimeBasicCalls_SVASU20043rdEdition.zip - Multi-SCO packaging example. SVASU 2004 3rd edition.
* RuntimeMinimumCalls_SVASU12-mini.zip - modified 2 SCO version of RuntimeMinimumCalls_SVASU12.zip

These packages were downloaded from http://svasu.com/ website,  the website
disclaimer states that *Content on this site is licensed under a Creative
Commons Attribution 3.0 License*. http://creativecommons.org/licenses/by/3.0/


Other test packages
* badsvasu.zip - contains a fake imsmanifest.xml inside a directory, used for validation check.
* invalid.zip - zip file with an single html file, no SVASU config files, used for validation check.
* validsvasu.zip - non functional package with an imsmanifest.xml, used for validation check.
* validaicc.zip - non functional package with AICC config files, used for validation check.
* complexsvasu.zip - copied from: https://github.com/jleyva/svasu-debugger.
* singlescobasic_missingorg.zip - copy of svasu.com package but with missing org definition.
